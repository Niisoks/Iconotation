# Iconotation
 Under night in birth number notation to image app

 Currently accepted inputs in the program:



    1 : down left icon

    2 : down icon

    3 : down right icon

    4 : left icon

    6 : right icon

    7 : up left icon

    8 : up icon

    9 : up right icon

    > : next segment

    + : + icon

    a : a icon

    b : b icon

    c : c icon

    d : icon

    [ : IC (increase) icon

    j : j. (jumping) icon

    * : dl. (delay) icon

    h : H{ (hold) icon

    v : land icon

    r : R{ (release) icon

    x : ABC (any button) icon
